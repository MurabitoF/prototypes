## Getting Started

Install the dependencies

```bash
  npm install
  yarn install
  pnpm install
```

Run in dev mode

```bash
  npm run dev
  yarn dev
  pnpm dev
```
## Tech Stack

**Client:** React, Zustand, Vite, Typescript
